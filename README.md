## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/react-projects.git
```

### 2️⃣ Navigate to the Project

```bash
cd React_Projects
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Start Development Server

```bash
npm run dev
```

### 5️⃣ Open in Browser

```text
http://localhost:5173
```
